# 💵 Conversor de Dólar Estático

Este é um projeto prático de nível iniciante desenvolvido em **C#** para consolidar conhecimentos fundamentais de lógica de programação, manipulação de variáveis e entrada/saída de dados via console.

## 📝 Sobre o Projeto
O programa permite que o utilizador insira um valor em dólares e receba a conversão correspondente em Reais (BRL). Atualmente, o projeto utiliza uma taxa de câmbio fixa (hardcoded) para fins de estudo da sintaxe

## 🚀 Tecnologias Utilizadas
- **Linguagem:** C#
- **IDE:** Visual Studio

## ⚙️ Funcionalidades
- Leitura de dados via teclado com `Console.ReadLine()`.
- Cálculo de conversão monetária.
- Formatação de saída numérica para duas casas decimais.
